# LazyBedrockBreaker
Minecraft基岩版一键破基岩辅助，基于[CoralFans](https://github.com/CoralFans-Dev/CoralFans)与[LegacyScriptEngine](https://github.com/LiteLDev/LegacyScriptEngine)。

## 兼容性
Minecraft Bedrock Edition 1.21.93
其他版本未测试

## 语言
仅简体中文

## 如何安装
1. 安装CoralFans([官方教程](https://coralfans-dev.github.io/CoralFans-doc/#/InstallDoc))与LegacyScriptEngine([官方教程](https://lse.levimc.org/zh/))
2. 下载 [最新Release](https://github.com/Eragon327/LazyBedrockBreaker/releases)
3. 解压，将整个文件夹放到服务器 'plugins' 目录下
4. 重启服务器，或在服务器控制台输入 'll load LazyBedrockBreaker'
